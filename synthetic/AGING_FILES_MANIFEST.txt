================================================================================
USAGE PATTERNS
================================================================================

1. Quick Age Single Image:
   from age_effects import age_map
   aged = age_map(img, intensity=0.6, style="1900", seed=42)

2. Custom Parameters:
   custom = {"yellowing": 0.7, "stains": 0.2}
   aged = age_map(img, custom_params=custom)

3. Batch Processing:
   from age_effects import batch_age_maps
   batch_age_maps(files, "output/", parallel=True)

4. Generate Training Data:
   for i, tile in enumerate(tiles):
       aged = age_map(tile, style=random.choice(styles), seed=i)
       aged.save(f"training/{i}.png")
